== Description ==

Post to DOCX allows to download post in DOCX format.

== Installation ==

1. Upload to the "post-to-docx" directory to `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use the community version included within the plugin or copy your PRO, PRO+, Corporate or Enterprise package to phpdocx folder
